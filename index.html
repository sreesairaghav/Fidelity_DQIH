Interactive Detailed Design

L0 Metadata Discovery & Coverage Intelligence

Discover schemas, tables, columns, datatypes, owners, timestamps and row counts.
Create central registry.
Calculate coverage score.
Detect tables without DQ rules.
Output: Asset Registry + Coverage Dashboard.

L1 Table Health Profiling

Checks: Row count trends, freshness, duplicate volume, partition effectiveness.
Stores historical snapshots.
Detects abnormal volume spikes and ETL failures.

L2 Column Health Profiling

Null %, cardinality, min/max, avg length, patterns, top values, skew detection, robust outlier detection.
Generates column quality score.

L3 Rule Validation Engine

Primary key uniqueness, composite keys, allowed values, date validations, custom YAML rule framework.
Failure severity assignment.
Failed records repository.

L4 Relationship Intelligence Graph

Automatically discovers table relationships.
Uses name matching, datatype matching and overlap analysis.
Builds dependency graph.
Validates referential integrity.
Supports impact traversal.

L5 Drift Intelligence

Volume drift, null drift, cardinality drift and distribution drift.
Statistical baseline generation.
No AI or ML used.
Uses historical profiling snapshots.

Impact Radius Engine

Identifies downstream tables, dashboards, reports and pipelines impacted by an issue.
Calculates blast radius and business impact.

Root Cause Correlation

Collapses multiple alerts into a single root cause.
Reduces alert fatigue.
Speeds up remediation.

Priority & Trust Scoring

Priority Score = Severity + Criticality + Impact Radius + Trend Risk.

Trust Score = Completeness + Validity + Integrity + Freshness + Stability.
Generates executive-level confidence metrics.
